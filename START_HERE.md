## 🐛 Troubleshooting

**Port sudah dipakai?**
```bash
# Matikan proses di port 3000/3001
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Jalankan ulang
npm run dev
```

**Animasi tidak jalan?**
- Refresh browser (Ctrl+F5)
- Clear cache
- Restart dev server
